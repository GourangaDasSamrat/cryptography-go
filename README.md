## Running Tests

Use the `justfile` to run tests for a specific chapter and level:

```bash
just test <chapter> <level>
```

**Examples:**

- `just test 1 1` - Run tests for Chapter 1, Level 1
- `just test 1 4` - Run tests for Chapter 1, Level 4
